# Swift Talk

Es una aplicación de mensajería en tiempo real, la cual permite a los usuarios enviar mensajes de texto e imágenes. La aplicación cuenta con un sistema de autenticación de usuarios, donde los usuarios pueden registrarse con su correo electrónico y contraseña.

## Listado de funcionalidades

- **Autenticación de usuarios**: Registro e inicio de sesión con correo electrónico y contraseña.
- **Mensajería en tiempo real**: Envío y recepción de mensajes de texto e imágenes en tiempo real.
- **Notificaciones push**: Notificaciones en tiempo real para nuevos mensajes.
- **Lista de contactos**: Los usuarios pueden ver una lista de sus contactos y buscar nuevos usuarios por su correo electrónico.
- **Chats individuales**: Los usuarios pueden iniciar chats individuales con sus contactos.
- **Carga de imágenes**: Los usuarios pueden enviar imágenes en los chats.
- **Diseño responsivo**: La aplicación está optimizada para funcionar bien en diferentes tamaños de pantalla.
- **Almacenamiento en la nube**: Uso de Firebase Firestore para almacenar mensajes y Firebase Storage para almacenar imágenes.
- **Optimización de rendimiento**: Uso de bibliotecas como Glide para la carga eficiente de imágenes.
- **Soporte para dispositivos Android**: Compatible con una amplia gama de dispositivos Android.
